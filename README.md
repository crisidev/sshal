SSH Launcher - SSHal

Requirements:
* python
* python-yaml
* pythondialog (https://github.com/turnkeylinux/pythondialog)
* python-blessings

Installation
Copy .sshal.yaml in your $HOME and edit it
Link sshal.py into /usr/local/bin
Launch sshal usage 

sshal.sh is a old version, here only for backup purpose
